# JavaScript Knowledge Tests

This folder contains various tests to assess JavaScript knowledge across different difficulty levels and topics.

## Test Categories

1. **beginner-fundamentals-test.js** - Tests covering variables, data types, operators, conditionals, loops, and basic functions
2. **intermediate-concepts-test.js** - Tests on arrays, objects, closures, prototypes, classes, and advanced function concepts
3. **async-programming-test.js** - Tests for promises, async/await, callbacks, and error handling
4. **dom-and-events-test.js** - Tests for DOM manipulation, event handling, and browser APIs
5. **advanced-topics-test.js** - Tests on modules, regex, network requests, storage, and web components
6. **coding-challenges/** - Practical coding challenges to solve
7. **multiple-choice-questions.md** - Multiple choice questions for quick assessment

## How to Use

### Running the Tests

Most test files can be run directly with Node.js:

```bash
node beginner-fundamentals-test.js
node intermediate-concepts-test.js
node async-programming-test.js
```

For DOM-related tests, open the HTML file in a browser or use a testing environment like Jest with jsdom.

### Assessing Knowledge

- **Beginner**: Complete the fundamentals test with 80%+ accuracy
- **Intermediate**: Pass intermediate concepts and start async programming tests
- **Advanced**: Successfully complete all tests including coding challenges

## Answer Keys

Each test file includes answers at the bottom (commented out) or in separate answer files.
