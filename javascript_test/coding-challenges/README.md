# Coding Challenges

This folder contains practical coding challenges to test JavaScript skills at different levels.

## Challenge List

### Beginner Challenges

1. **fizzbuzz.js** - Classic FizzBuzz problem
2. **array-sum.js** - Calculate sum of array elements
3. **palindrome-checker.js** - Check if a string is a palindrome
4. **find-max.js** - Find the maximum value in an array
5. **reverse-string.js** - Reverse a string

### Intermediate Challenges

1. **deep-flatten.js** - Flatten nested arrays
2. **debounce-function.js** - Implement debounce utility
3. **promise-all.js** - Implement Promise.all from scratch
4. **array-grouping.js** - Group array elements by a key
5. **object-deep-clone.js** - Deep clone an object

### Advanced Challenges

1. **event-emitter.js** - Implement an EventEmitter class
2. **lru-cache.js** - Implement LRU Cache
3. **async-queue.js** - Implement async task queue with concurrency limit
4. **observable.js** - Implement Observable pattern
5. **json-parser.js** - Parse JSON without using JSON.parse()

---

## How to Use

Each challenge file contains:
- Problem description
- Function signature to implement
- Example inputs/outputs
- Test cases to verify your solution

Run the tests with:
```bash
node fizzbuzz.js
```

## Scoring

- **Beginner**: Complete 4/5 challenges correctly
- **Intermediate**: Complete 3/5 challenges correctly  
- **Advanced**: Complete 2/5 challenges correctly

## Tips

1. Read the problem carefully
2. Write tests before implementing (TDD approach)
3. Consider edge cases
4. Optimize for both time and space complexity
5. Write clean, readable code with comments
