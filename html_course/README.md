# HTML Basics Course

Welcome to the comprehensive HTML course. This course is designed to provide you with a solid foundation in web structure, taking you from the basics of HTML to advanced semantic elements and accessibility best practices.

## Course Roadmap

The course is divided into four main modules, covering everything from initial setup to creating accessible, well-structured web pages.

### [Module 1: Getting Started & Structure](./01-getting-started-and-structure.md)
- **Stage 1: Getting Started:** What HTML is, how browsers work, and writing your first page.
- **Stage 2: Document Structure:** DOCTYPE, `<html>`, `<head>`, `<body>`, meta tags, and the title element.

### [Module 2: Text, Links & Images](./02-text-links-and-images.md)
- **Stage 3: Text & Headings:** H1 to H6, paragraphs, line breaks, strong, em, and inline text elements.
- **Stage 4: Links & Images:** Anchor tags, href, target, relative vs absolute paths, img, alt text, and src.
- **Stage 5: Lists:** Ordered lists, unordered lists, list items, and nested lists.

### [Module 3: Tables & Forms](./03-tables-and-forms.md)
- **Stage 6: Tables:** Table, thead, tbody, tr, th, td, colspan, and rowspan.
- **Stage 7: Forms:** Form element, input types, labels, buttons, select, textarea, and form attributes.

### [Module 4: Semantic HTML & Accessibility](./04-semantic-html-and-accessibility.md)
- **Stage 8: Semantic HTML:** Header, nav, main, article, section, aside, footer and why they matter.
- **Stage 9: Accessibility:** ARIA roles, alt attributes, lang attribute, valid HTML, and validation tools.
