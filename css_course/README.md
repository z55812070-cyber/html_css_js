# CSS Basics Course

Welcome to the comprehensive CSS course. This course is designed to take you from the fundamentals of styling web pages to advanced layout techniques and animations.

## Course Roadmap

The course is divided into five main modules, covering everything from basic syntax to creating responsive, interactive web designs.

### [Module 1: Fundamentals & Box Model](./01-fundamentals-and-box-model.md)
- **Stage 1: CSS Fundamentals:** What CSS is, linking to HTML, and the cascade.
- **Stage 2: Selectors & Specificity:** Element, class, ID, pseudo-classes, and combinators.
- **Stage 3: Box Model:** Content, padding, border, margin, and box-sizing.

### [Module 2: Typography, Color & Positioning](./02-typography-color-and-positioning.md)
- **Stage 4: Typography & Color:** Font families, sizing, weights, color values, and backgrounds.
- **Stage 5: Display & Positioning:** Block vs inline, relative, absolute, fixed, and sticky positioning.

### [Module 3: Flexbox & Grid](./03-flexbox-and-grid.md)
- **Stage 6: Flexbox:** Containers, alignment, wrapping, and item properties.
- **Stage 7: CSS Grid:** Templates, rows, columns, gaps, and placement.

### [Module 4: Responsive Design & Animations](./04-responsive-design-and-animations.md)
- **Stage 8: Responsive Design:** Media queries, breakpoints, and relative units (rem, em, vw, vh).
- **Stage 9: Transitions & Animations:** Keyframes, transforms, and easing.

### [Module 5: Variables, Organization & Project](./05-variables-organization-and-project.md)
- **Stage 10: Variables & Organization:** Custom properties, naming conventions, and BEM methodology.
- **Stage 11: Practice Project:** A final project to build a responsive card component.
