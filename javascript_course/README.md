# Modern JavaScript Course

Welcome to the comprehensive JavaScript course, inspired by [javascript.info](https://javascript.info/). This course is designed to take you from a complete beginner to an advanced JavaScript developer.

## Course Roadmap

The course is divided into five main modules, covering the language core, browser interaction, and advanced specialized topics.

### [Module 1: Beginner Fundamentals](./01-beginner-fundamentals.md)
- **Introduction:** What is JavaScript, editors, and the developer console.
- **JavaScript Fundamentals:** Variables, data types, operators, conditionals, loops, and functions.
- **Objects Basics:** Introduction to objects, references, methods, and the 'new' operator.

### [Module 2: Intermediate Language](./02-intermediate-language.md)
- **Data Types in Depth:** Numbers, strings, arrays, maps, sets, and JSON.
- **Advanced Functions:** Recursion, rest/spread, closures, decorators, and function binding.
- **Prototypes & Classes:** Prototypal inheritance and the modern class syntax.

### [Module 3: Async and Advanced JS](./03-async-and-advanced-js.md)
- **Error Handling:** try...catch and custom errors.
- **Asynchronous JavaScript:** Callbacks, Promises, and Async/Await.
- **Language Features:** Generators, modules, and miscellaneous features (Proxy, Reflect).

### [Module 4: Browser: DOM and Events](./04-browser-dom-and-events.md)
- **The Document (DOM):** DOM tree, navigation, and modifying the document.
- **Events:** Introduction to events, bubbling/capturing, and event delegation.
- **UI Events:** Mouse, keyboard, and scroll events.
- **Forms:** Form properties, methods, and submission.

### [Module 5: Additional Advanced Topics](./05-additional-advanced-topics.md)
- **Network Requests:** Fetch API, FormData, and URL objects.
- **Data Storage:** Cookies, LocalStorage, and IndexedDB.
- **Specialized Topics:** Regular expressions, Web Components, and Animation.
