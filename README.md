# Web Development Courses

Welcome to our comprehensive web development learning platform. Here you will find structured courses for HTML, CSS, and JavaScript, designed to take you from a beginner to an advanced developer.

## Available Courses

### [HTML Basics Course](./html_course/README.md)
Learn the foundations of web structure, from basic tags to semantic HTML and accessibility best practices.

### [CSS Basics Course](./css_course/README.md)
Master the art of styling web pages, covering everything from the box model and typography to advanced layouts with Flexbox and Grid.

### [Modern JavaScript Course](./javascript_course/README.md)
Dive into the core of web interactivity, from beginner fundamentals to advanced asynchronous programming and browser interaction.
