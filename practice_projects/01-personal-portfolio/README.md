# Personal Portfolio Website

A responsive, modern personal portfolio website to showcase your projects and skills.

## 🎯 Learning Objectives

- Semantic HTML structure
- CSS Flexbox and Grid layouts
- Responsive design with media queries
- CSS custom properties (variables)
- Smooth scrolling and transitions
- Mobile-first design approach

## 📋 Features

- Hero section with introduction
- About me section
- Skills display with visual indicators
- Project gallery with hover effects
- Contact form (static)
- Fully responsive navigation
- Dark/Light theme toggle

## 🛠️ Concepts Applied

### HTML
- Semantic elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)
- Proper heading hierarchy
- Accessible forms
- Image optimization with `alt` attributes

### CSS
- CSS Variables for theming
- Flexbox for navigation and card layouts
- CSS Grid for project gallery
- Media queries for responsiveness
- Transitions and transforms for animations
- Fixed positioning for navigation
- BEM naming convention

## 📁 Files Included

- `index.html` - Complete HTML structure
- `style.css` - All styles with variables and responsive breakpoints
- `script.js` - Theme toggle and mobile menu functionality
- `assets/` - Placeholder for images

## 🚀 Getting Started

1. Open `index.html` in your browser
2. Customize the content with your information
3. Replace placeholder images with your own
4. Deploy to GitHub Pages, Netlify, or Vercel

## 💡 Customization Ideas

- Add a blog section
- Include testimonials
- Add social media links with icons
- Create a timeline of your experience
- Add project filters

## 🎨 Color Scheme

The project uses CSS variables for easy theming:

```css
:root {
  --primary-color: #6366f1;
  --secondary-color: #8b5cf6;
  --text-dark: #1f2937;
  --text-light: #6b7280;
  --bg-light: #ffffff;
  --bg-dark: #111827;
}
```

## 📱 Responsive Breakpoints

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
