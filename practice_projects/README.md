# Practice Projects with Solutions

This directory contains portfolio-worthy practice projects designed to reinforce concepts from the HTML, CSS, and JavaScript courses. Each project increases in complexity and showcases different skills.

## Project List

### 1. **Personal Portfolio Website** (Beginner)
- **Focus:** HTML structure, CSS styling, responsive design
- **Concepts Used:** Semantic HTML, Flexbox/Grid, Media Queries, CSS Variables
- **Live Demo:** [View Solution](./01-personal-portfolio/)

### 2. **Task Manager App** (Intermediate)
- **Focus:** DOM manipulation, event handling, local storage
- **Concepts Used:** DOM selection/modification, Event listeners, LocalStorage, Array methods
- **Live Demo:** [View Solution](./02-task-manager-app/)

### 3. **Weather Dashboard** (Intermediate-Advanced)
- **Focus:** Async JavaScript, API integration, error handling
- **Concepts Used:** Fetch API, Promises/Async-Await, Error handling, JSON parsing
- **Live Demo:** [View Solution](./03-weather-dashboard/)

### 4. **Recipe Finder** (Advanced)
- **Focus:** Complex state management, search functionality, filtering
- **Concepts Used:** Advanced array methods, Debouncing, Dynamic UI rendering, Form handling
- **Live Demo:** [View Solution](./04-recipe-finder/)

### 5. **Real-time Chat Application** (Expert)
- **Focus:** WebSocket communication, real-time updates, advanced UI
- **Concepts Used:** WebSocket API, Event delegation, Advanced DOM manipulation, CSS animations
- **Live Demo:** [View Solution](./05-chat-application/)

## How to Use These Projects

1. **Study First:** Review the relevant course modules before attempting each project
2. **Try Yourself:** Attempt to build the project without looking at the solution
3. **Compare:** Review the solution code to learn best practices
4. **Customize:** Add your own features and styling to make it unique for your portfolio
5. **Deploy:** Host your projects on platforms like Netlify, Vercel, or GitHub Pages
